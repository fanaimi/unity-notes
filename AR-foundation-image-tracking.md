# unity-notes - AR FOUNDATION - IMAGE TRACKING

* AR Session Origin > add compoment > AR Tracked Image Manager
* the Tracked Image manager needs 2 thingd:
	* the prefab to instantiate
	* a library of images to track

* create a folder for AR images
	* save a few trackable images in the folder

* in the ARimages folder > create > XR > Reference Image Library
	* in the ReferenceImageLibrary, add > drag an image, repeat	
	* for iOS it's mandatory to specify image size	

* drag the ReferenceImageLibrary in AR session origin > serialised library
