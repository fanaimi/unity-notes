# unity-notes - VUFORIA

* go tot VUFORIA developer portal -> https://developer.vuforia.com/ -> create an account and log in 

* got to TARGET MANAGER -> ADD DATABASE 
	* type DB name and set type = device

* photograph two flat and complicated objects to be the targets, import images in pc

* upload images to DB, click on DB -> Add Target
	* select single image under target type
	* make sure targets have at least 3 stars
	* select size (unit is meters)
 
* Download Database -> Unity Editor	

* License Manager -> Get Basic -> give the license a name, click on confirm

* go to Downloads -> "Add Vuforia Engine to a Unity Project or update to latest version"
